hub.port – Порт¶

Модуль содержит экземпляры класса Port, которые создаются автоматически при старте хаба. Создать их самостоятельно невозможно. Каждому из экземпляров можно обратиться в соответствии с его названием:
hub.port.A, hub.port.B, hub.port.C, hub.port.D, hub.port.E, hub.port.F

Класс Port¶

Предоставляет доступ к конфигурации порта, а также устройства подключенного к этому порту.

Методы, доступные при любом режиме работы

pwm(value: int) → None¶

Подает ШИМ сигнал на силовые выходы порта или устройства.

Если установить значение равно 0, результат для моторов будет аналогичен команде float(). Т.е. свободное вращение без торможения.

Параметры

value – ШИМ значение от -100 до +100. Полярность прикладываемого сигнала соответствует знаку задаваемой величины. Значение равное 0 отключает ШИМ сигнал.

Пример кода
from hub import port
port.A.pwm(50)

Константы¶

События порта

Эти константы могут быть использованы с методом Port.callback.

hub.port.DETACHED = 0¶

Устройство отключено от порта.

hub.port.ATTACHED = 1¶

Новое устройство было подключено к порту.

Режимы работы

Эти константы могут быть использованы с методом Port.mode.

hub.port.MODE_DEFAULT = 0¶

Режим для устройств совместимых с Powered Up протоколом.

hub.port.MODE_FULL_DUPLEX = 1¶

Полнодуплексный режим работы.

hub.port.MODE_HALF_DUPLEX = 2¶

Полудуплексный режим работы.

hub.port.MODE_GPIO = 3¶

Режим работы на ввод и вывод используя Pin.
