Hub Documentation¶

Представляем Вам документацию MicroPython для LEGO MINDSTORMS Inventor Hub. MicroPython легкая и эффективная реализация языка Python, которую можно использовать на LEGO MINDSTORMS Inventor Hub

LEGO расширила MicroPython пакетом hub, позволяющем получить доступ к моторам, сенсорам и датчикам. На этих страницах Вы найдете документацию для пакета hub. Используйте меню слева для навигации по всем доступным модулям этого пакета.
